CSS CHEAT SHEET

Function Name Attributes and Format Description

Background Color

 selector {background-color:purple;} Changes the background color of an element.

Text Color

 selector{color:blue;}
selector{color:#0208b0}
selector{ color: hsl(238, 98, 35);}
selector {color: rgb(2, 8, 176);} 		Color can be used as names, hex codes, hsl, and rgb. Names are least precise.

Alpha Colors

 transparency with rgba and hsla Alpha values determine the transparency of colors in CSS. Alpha values can be set for both RGB and HSL colors by using rgba() and hsla() and providing a fourth value representing alpha. Alpha values can range between 0.0 (totally transparent) and 1.0 (totally opaque).

Font Weight

 selector{font-weight:400;} Thickness of font. Numerical values can be used. Ranges from 100 to 900 and accepts only multiples of 100. Default numerical value is 400. Less than 400 = lighter text, greater than 400 = bolder. The provided value can also be a keyword such as "bold" or "normal".

Text Align

 selector{text-align: right;} The text-align CSS property can be used to set the text alignment of inline contents. This property can be set to these values: left, right, or center

Line Height

 selector{line-height: 10px;} Shifts space between lines (vertically, not between letters which is "letter-spacing")

Margin (horizontal alignment)

 selector{margin: auto;} The value AUTO can be used with the property margin to horizontally center an element within its container. The margin property will take the width of the element and will split the rest of the space equally between the left and right margins.
